Description Spécifique du Projet de Gestion Scolaire 1. Génération Automatique d'Email Fonctionnalité : Lors de l'ajout d'un nouvel étudiant ou enseignant, un email est généré automatiquement. Format de l'Email : Pour les étudiants : initiale_du_prenom.nom@edu.umi.ac.ma Exemple : Pour un étudiant nommé "Ilys Benrd", l'email généré sera i.benrd@edu.umi.ac.ma. Si l'email existe déjà, un compteur est ajouté pour créer un nouvel email unique (ex. i.benrd1@edu.umi.ac.ma) pour les enseignants meme chose juste on supprime le mot edu. 2. Menus par Rôle a. Menu Administrateur Fonctionnalité : Permet à l'administrateur de gérer les étudiants, enseignants, matières et emploi du temps. Options du Menu : Gérer les Étudiants : Ajouter, modifier, supprimer ou afficher la liste des étudiants. Gérer les Enseignants : Ajouter, modifier, supprimer ou afficher la liste des enseignants. Gérer les Matières : Ajouter, modifier, supprimer ou afficher la liste des matières. Gérer l'Emploi du Temps : Générer un emploi du temps automatique, ajouter, modifier ou supprimer des créneaux. Quitter : Déconnexion de l'application. b. Menu Enseignant Fonctionnalité : Permet aux enseignants de gérer les notes des étudiants et d'accéder à leurs informations. Options du Menu : Affecter des Notes : Ajouter, modifier ou supprimer des notes pour les étudiants. Afficher la Liste des Étudiants : Afficher tous les étudiants inscrits dans la classe ou la matière enseignée. Quitter : Déconnexion de l'application. c. Menu Étudiant Fonctionnalité : Permet aux étudiants de consulter leurs informations et résultats. Options du Menu : Consulter Mes Notes : Afficher les notes de l'étudiant pour chaque matière. Consulter Mes Modules : Afficher les matières dans lesquelles l'étudiant est inscrit. Afficher Mon Emploi du Temps : Afficher l'emploi du temps personnel de l'étudiant. Afficher les Annonces : Consulter les annonces publiées par les enseignants ou l'administration. Quitter : Déconnexion de l'application. 3. Gestion des Étudiants Fonctionnalité : Permet de gérer les informations des étudiants. Détails : Chaque étudiant a un identifiant unique, un nom, un prénom, un email, un CNE (numéro d'étudiant), une section et une filière. Les informations des étudiants sont stockées dans un fichier etudiants.txt. 4. Gestion des Enseignants Fonctionnalité : Permet de gérer les informations des enseignants. Détails : Chaque enseignant a un identifiant unique, un nom, un prénom, un email, un code enseignant et une matière enseignée. Les informations des enseignants sont stockées dans un fichier enseignants.txt. 5. Gestion des Matières Fonctionnalité : Permet de gérer les matières enseignées. Détails : Chaque matière a un identifiant unique, un code matière, un nom de matière et un coefficient. Les informations des matières sont stockées dans un fichier matieres.txt. 6. Gestion des Notes Fonctionnalité : Permet de gérer les notes des étudiants pour chaque matière. Détails : Les notes comprennent les devoirs surveillés (DS1, DS2) et les travaux pratiques (TP). Les informations des notes sont stockées dans un fichier notes.txt. 7. Gestion de l'Emploi du Temps Fonctionnalité : Permet de gérer l'emploi du temps de l'école. Détails : L'emploi du temps est constitué de créneaux horaires, chaque créneau ayant un jour, une heure, une matière, un enseignant et une section. Les informations de l'emploi du temps sont stockées dans un fichier emploi_du_temps.txt. 8. Authentification et Sécurité Fonctionnalité : Assurer la sécurité des données et des accès. Détails : Les utilisateurs doivent entrer un email et un mot de passe pour accéder à leurs menus respectifs. Les mots de passe sont stockés de manière sécurisée (idéalement hachés).
